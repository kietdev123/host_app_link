📋 Hướng dẫn setup

  1. Thay thế com.example.yourapp bằng package name thật của app
  2. Thay thế SHA256 fingerprint bằng fingerprint thật của app
  3. Cập nhật domain trong file assetlinks.json
  4. Enable GitHub Pages cho repository này

🔧 Lấy SHA256 Fingerprint

Sử dụng lệnh sau để lấy SHA256 fingerprint:

keytool -list -v -keystore your-release-key.keystore

📁 File assetlinks.json

File này được đặt tại: /.well-known/assetlinks.json
